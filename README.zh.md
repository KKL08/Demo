## 🌸 赛博圣地巡礼 (Cyber Anime Pilgrimage)

> Gemini 驱动的动漫旅行影像生成器 — 足不出户，让你的推跨越次元壁打卡现实世界。

上传一张角色图片并输入番剧名称，AI 会自动考据该番剧在现实世界中的取景地（圣地），并生成一张角色在真实场景中的旅行照。

**核心功能**
- 🗺️ AI 圣地考据 — 自动检索番剧真实取景地及背景故事
- 📸 跨次元合影生成 — 将二次元角色融入三次元现实风景
- 💾 本地历史记录 — 自动保存最近 5 次巡礼足迹
- ⬇️ 一键下载生成的旅行照
- 🔐 隐私安全优先 — API Key 仅在本地使用，不上传任何服务器

**使用前提：** 需要 Google Gemini API Key（[免费获取](https://aistudio.google.com/apikey)）

**使用步骤：**
1. 🔑 输入你的 Google Gemini API Key
2. 🖼️ 上传动漫角色图片（背景尽量干净）
3. 📝 输入番剧名称（选填角色名可提高准确度）
4. ✨ 点击"开始寻找圣地" → 选择地点 → 点击"生成旅行照"
